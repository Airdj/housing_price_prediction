# housing_price_prediction
Project is based on the use of machine learning models for predicting real estate prices through analysis and selecting factors important for model optimization.

Kaggle competition link: https://www.kaggle.com/competitions/home-data-for-ml-course/overview

Score is evaluated on Root-Mean-Squared-Error (RMSE) between the logarithm of the predicted value and the logarithm of the observed sales price. (Taking logs means that errors in predicting expensive houses and cheap houses will affect the result equally.)

Actual score(RMSE): 13678.95078
